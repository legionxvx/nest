get(name)¶

Return the value at key name, or None if the key doesn’t exist

set(name, value, ex=None, px=None, nx=False, xx=False)¶

Set the value at key name to value

ex sets an expire flag on key name for ex seconds.

px sets an expire flag on key name for px milliseconds.

nx if set to True, set the value at key name to value only

if it does not exist.

xx if set to True, set the value at key name to value only

if it already exists.

get_products(filter=[], blacklist=[], *args, **kwargs)¶

Yields product name, price, and alias information. Useful for updating the corresponding database model.

session = FastSpring()
session.get_products(
    filter=["ava-ds", "ava-mc],
    blacklist=["bundles"]
)
Parameters

  • filter – Whitelisted product ids.

  • blacklist – Blacklisted product

  • args – Other positional arguments passed to each GET request.

  • kwargs – Other keyword arguments passed to each GET request.

property prefix¶

URL to prefix on all requests. Since this is dedicated to FastSpring, it is (currently): ‘https://api.fastspring.com’.

This property is read-only and has no setter.

Parameters

  • suffix – Joined to prefix.

  • args – Other positional arguments passed to request.

  • kwargs – Other keyword arguments passed to request.

property default_list¶

Since most of this API usage is dedicated to maintaining and modifying one list, this property acts as a default for requests that do not specify a list id.

This property is the name of the list, which will be extracted from lists.

Like urllib.parse.urljoin(), except it takes a variable number of args and a variable seperator.

  • suffix – Joined to prefix.

  • args – Other positional arguments passed to request.

  • kwargs – Other keyword arguments passed to request. If list is included in kwargs, it will popped and used in place of the default_list

property recipients¶

The list of users for whom these items are propogated to.

Since FastSpring allows gifting of purchases, this list could, theoretically, be infinitely long. However, for the time being, we assume that the first recipient is the actual intended recipient of this purchase. Typically, this is also the customer.

class nest.apis.fastspring.events.WebhookEvent(data={}, session=None, type_hint=None)¶

Base class for all webhook events of a FastSpring API Webhook event.

If a session is provided, most of the properties will be valid new or existing database objects. Otherwise, the properties are the parsed webhook event data.

abstract convert(subclass)¶

Convert this WebhookEvent into a more refined subclass.

Data and database session are passed to subclass’ constructor.

Parameters

subclass – The derived class of a WebhookEvent.

abstract is_order()¶

True if webhook type is 'order.completed'.

abstract is_return()¶

True if webhook type is 'return.created'.

abstract is_subscription_activated()¶

True if webhook type is 'subscription.activated'.

abstract is_subscription_deactivated()¶

True if webhook type is 'subscription.deactivated'.

abstract property model¶

The resulting database object, if any.

abstract to_order()¶

Convert this WebhookEvent to an Order.

abstract to_return()¶
